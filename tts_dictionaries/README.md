# tts_dictionaries

TTS（Irodori）読み上げ用の「英字 → カナ」変換辞書を置くフォルダです。

## 目的

Irodori-TTS は日本語音声モデルのため、ラテン文字（英単語）を読み上げると生成が暴走し、
意味不明な音声になることがあります。そこで **TTS へ渡す直前だけ** 英字を **カナ（発音）** へ
置き換えます。会話ログ・画面表示は **原文のまま**（英字を残す）で、LM の理解にも影響しません。

## 辞書ファイルの形式

- 文字コード: UTF-8
- 1 行 1 エントリ: `english,カタカナ`
  - 区切りはカンマ `,` またはタブ
  - 先頭 `#` の行はコメント、空行は無視
  - 英字キーは大文字小文字を区別しない
- [alkana](https://github.com/cod-sushi/alkana) の `alkanadict.csv`（`word,kana` 形式）を
  そのまま置いて使うこともできます（pip 依存は追加しません。辞書ファイルのみ利用）。

## 環境変数

| 変数 | 既定 | 説明 |
| --- | --- | --- |
| `TTS_KANA_NORMALIZE` | `on` | `on`/`off`。`off` で変換を無効化（従来どおり原文を TTS へ）。 |
| `TTS_KANA_DICT_FILE` | `tts_kana_dict.csv` | 使用する辞書ファイル。`alkanadict.csv`（`tts_dictionaries/` 配下）／`tts_dictionaries/alkanadict.csv`（プロジェクト相対）／絶対パス のいずれも可。`;` 区切りで複数指定でき、**後に書いたファイルが優先**（例 `alkanadict.csv;tts_kana_dict.csv`）。 |

## フォールバックと安全性

辞書に無い語は自動でカナへ近似変換し、さらに最終ガードで残ったラテン文字を必ずカナへ
落とします。したがって辞書が空でも Irodori にラテン文字が渡ることはなく、暴走しません。

辞書の値は必ず **カタカナの読み（発音）** を入れてください（意味の日本語訳は入れない）。
たとえば `dysentery` は英語の音「ディセントゥリー」で読ませます。`赤痢` のように訳語を
入れると別語を読み上げてしまい、漢字の誤読も招きます。用途は「alkana の音写が不自然な語の
上書き」や「alkana に無い略語・新語（PCR, COVID など）の読み追加」です。
